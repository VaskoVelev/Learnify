# Learnify – Online Learning Platform

## Overview

Learnify is a modern web-based platform for online learning, developed as a diploma project. The system allows teachers to create and manage courses, and students to enroll and participate in them. The platform provides secure authentication, role-based access, and a user-friendly interface.

The goal of the project is to demonstrate the design and implementation of a full-stack web application using modern technologies and best practices.

---

## Main Features

* User registration and login with authentication
* Role-based access
* Browse and enroll in courses
* View enrolled courses
* Course management functionality
* Secure backend REST API
* Responsive frontend interface

---

## Technologies Used

### Frontend

* React
* JavaScript
* React Router
* Axios
* Tailwind CSS

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* REST API

### Database

* PostgreSQL

---

## Architecture

The application follows a client-server architecture:

* Frontend communicates with the backend via REST API
* Backend handles business logic, authentication, and database access
* Database stores users, courses, and enrollments
